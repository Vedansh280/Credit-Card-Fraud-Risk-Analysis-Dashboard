Credit Card Fraud & Risk Analysis Dashboard
About Project
This project is built using a Kaggle dataset containing detailed records of credit card transactions. An interactive dashboard was created in Power BI to analyze fraud types, risk levels, and state‑wise distribution.

Problem Analysis
High fraud rate observed across states like Maharashtra, Karnataka, and West Bengal.

Critical risk transactions affecting merchants such as Lifestyle, Tata Cliq, and Big Bazaar.

Fraud types including Card Skimming, Identity Theft, Account Takeover, and Phishing are recurring.

Seasonal spikes in November and December highlight increased fraud during festive seasons.

Problem Solved
Added filters (Fraud Type, State, Merchant Name) for quick fraud detection.

Risk segmentation (Low, Medium, High, Critical) helps classify transactions efficiently.

State‑wise and month‑wise charts identify fraud hotspots.

Business insights reveal vulnerable merchant categories like E‑commerce, Food Delivery, and Transportation.

Functions Used
Power BI DAX: SUM, COUNTROWS, CALCULATE, FILTER

Data Cleaning: Removed duplicates, formatted dates, handled blanks

Visualization: Donut chart, line chart, bar chart for comparative analysis
